# ToDo CLI app test

Test å lage en enkle ToDo app med ncurses for terminal

## Hurtigstart

```console
cargo run
```

## Kontroll
|Key|Beskrivelse|
|---|-----------|
|<kbd>q</kbd>|avslutt program|
|<kbd>w</kbd>|flytt opp i listen|
|<kbd>s</kbd>|flytt ned i listen|
|<kbd>enter</kbd>|flytt todo til fullført (ferdig)|
